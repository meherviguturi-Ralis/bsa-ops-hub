# BSA Ops Hub

AI-powered BSA workflow app for:

- Change request intake

- AI parsing

- ADO ticket creation

- Obsidian knowledge capture

- Release calendar

- Loan context

- MCP tool integration

- Manual and future Outlook-based intake

- Structured ticket generation

- Save knowledge notes to Obsidian

- Create ADO tickets

- Sprint/release tracking

- Loan context module
